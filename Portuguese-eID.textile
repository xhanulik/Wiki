h2. Card History

* 2007-02-14 First card version with a IAS 0.7 V1, EMV-CAP and Precise Bio MOC applets and 1024b keys.
* 2015-06-05 Owner keys size increased to 2048b.
* 2016-01-16 EMV-CAP applet is no longer present.

h3. Applets used

h2. Biometrics

All cards include a Precise Biometric Match-On-Card applet and the correspondent owner fingerprint template.

Documentation or sample code contribution is appreciated.

h2. Official Middleware
 * Official releases are available at: "Autenticação.gov.pt":https://www.autenticacao.gov.pt/cc-software
 * Source code is available at: "SVN.gov.pt":https://svn.gov.pt/projects/ccidadao

h2. Old Proprietary Middleware

The "software download page":https://www.cartaodecidadao.pt/index.php_option=com_content&task=view&id=102&Itemid=44&lang=pt.html listed software installers for Windows, Mac OS X and several Linux distributions. Parts of the software are based on OpenSC (custom libopensc provided with the software). Originally the Portuguese Government was violating the OpenSC GPL by refusing to provide the modified source files.

h2. OpenSC support

Starting with version 0.12.0, OpenSC supports the Portuguese eID card - both Authentication and Digital Signature keys.

Full Mac OS X support is available through OpenSC.Tokend.
